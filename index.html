Selected Works

Macro

Macro is an iOS macronutrient tracking app using computer vision to identify given food groups to provide users with relevant information on a food's amount of proteins, carbohydrates, and fats. Led the design and frontend implementation, synthesized user feedback and insights to create composable elements in React Native and Redux to display past meals. I also added a feature to query relevant recipes from the Spoonacular database providing the needed ingredients and steps.
